# Software-I

## Purpose of this project

This project is for a university course to demonstrate my knowledge of Java

## What are you looking at here?

This project's requirements were to, using exclusively Java and JavaFX, build out a GUI project that would implement a rudamentary inventory management system.

### Requirements:

- Use only JavaFX and no external libraries to build out the UI
- Allow the user to add (CRUD) parts to the system
- Allow the user to make (CRUD) products consisting of one or more parts that exist in the system
